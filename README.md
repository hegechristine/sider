# sider.hegechristine.no

Subdomene for sider, eksperimenter og innhold som ikke har funnet hjem ennå.

Hostes på GitHub Pages, midlertidig — flyttes til Kajabi når MCP-en er tilgjengelig.

## Oppsett

- DNS: CNAME `sider` → `hegechristine.github.io.` (Pro-ISP)
- Custom domain: `sider.hegechristine.no` (CNAME-fil i repo-rot)
- Designsystem: hegechristine.github.io/design-system v1.0

## Sider på subdomenet

- [/lenker/](lenker/) — link-in-bio
- [/visittkort/](visittkort/) — digitalt visittkort med vCard + skjema-automasjon ([detaljer](visittkort/README.md))
- [/150k/](150k/) — salgsside for 150K-utfordringen (+ alt-varianter og takk-sider)
- [/30-dager/](30-dager/) — affiliate-side, plattformen jeg bygger businessen på (30-dagers gratis trial)
- [/kajabi/](kajabi/) — redirect → /30-dager/ (gammel URL, byttet pga Partner Agreement §4.3)
